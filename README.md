# XAI_FER_SHAP

SHAP을 활용한 설명 가능한 얼굴 표정 인식


  이 프로젝트는 SHAP을 사용하여 얼굴 표정 인식 모델이 이미지를 어떻게 학습하고 어떤 기준으로 감정을 분류해내는지 인간이 이해할 수 있도록 설명하고자 한다. CNN 기반으로 얼굴 표정 인식 모델을 구현하고, 구현한 모델의 결과를 해석해내는 과정에서 설명 가능한 AI 기법 중 하나인 SHAP을 사용하였다. 그 결과 얼굴 표정 인식 모델이 감정을 분류할 때에는 얼굴 중에서도 눈과 입 주변 영역을 주요 감정 판단 근거로 삼는다는 사실을 확인하였다.
   SHAP은 분류 예측에 활용되는 모든 이미지 요소에 대하여 예측 결과에 기여하는 정도를 나타내는 Shapley value값을 도출하여 활용한다. 이에 따라 인공지능 모델이 학습하고 예측 결과를 제시하는 과정에서 각 요소의 기여도를 수치적으로 확인할 수 있다. 더 나아가, 본 논문에서는 이 Shapley value를 이용하여 noise data를 추출하고 이를 학습에 제외함으로써 결과적으로 학습 모델의 성능을 향상시키는 모델 구현 프로세스를 제안하였다.
